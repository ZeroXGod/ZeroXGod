Composer 学习笔记

什么是 Composer？

Composer 是 PHP 的一个依赖管理工具。

Compser 的作用

  • 引用代码到项目中
  • 管理引用的代码

Composer 的好处

  • 模块化，降低代码重用成本
  • 统一的第三方代码组织方式
  • 更科学的版本更新

Composer 工作原理

composer 下载时先到 Composer 官方数据源（Packagist）搜索包，获取包的数据。然后再到Github 下载包的对应代码。
